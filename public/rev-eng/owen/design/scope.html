02
Customer portal (your #1 pain → solved)
Weeks 4–7 ships in 28 days

Goal: eliminate “where's my watch?” comms forever. Replace 90% of FB Messenger / email back-and-forth with a single dashboard.

  • Customer login — passwordless (email magic link or passkey)
  • Order dashboard — every order with build #, status, ship window
  • Build timeline — 5 stages: Sourced → Assembled → Regulated → QC'd → Shipped, with photo/video at each
  • Locked spec card — case, dial, hands, bezel, strap, engraving, all in one immutable doc both sides reference
  • In-portal chat with Owen — threaded messaging per order, Owen sees a single inbox of all customers, push + email notifications, replaces Facebook Messenger entirely. This is the centralized comms system you asked for.
  • Build photo uploads — Owen uploads from phone, customer sees in real time
  • Build-stage email + SMS — automated at each timeline transition
  • Watch history — every past build with spec, serial, date, photos
  • Service / repair requests — “regulation tune-up,” “case refurbish,” “movement service” forms
  • Warranty status — active / expiring tracking per watch
  • Owen's admin view — single inbox, build queue, daily prep checklist, batch photo uploader
  • Public order page (optional) — shareable countdown link customers can post on IG

// things that could go wrong

Risks, named upfront.

Every risk has a mitigation we've already thought about. None of these are dealbreakers — but worth flagging so there are no surprises.

  • 01
    Seiko IP risk
    Seiko issued an Oct 2025 anti-mod notice targeting commercial sellers using counterfeit Seiko-branded parts. Mitigation: the brand evolves to one that doesn't lead with “Seiko” (TCT, Yohman, or bench/22 all qualify). Modded Seiko keeps the IG handle but uses “Seiko-powered” not “Seiko mod” in commercial copy. Genuine Seiko movements (NH35) are legal — only counterfeit dials/cases are an issue.
  • 02
    Photography quality cap
    Existing iPhone shots are good for a v0 launch but cap the premium positioning above $700. Mitigation: $1,500 half-day pro shoot in Phase 1, scoped on the shot list we'll deliver before greenlight. Optional but recommended.
  • 03
    Owen bandwidth as volume grows
    Building 60–100 watches/month + handling all comms is a real time crunch even with the portal. Mitigation: the portal eliminates ~70% of comms work. If volume grows past 50/month, plan a part-time assembly assistant ($15–20/hr, 2–3 days/week) by month 9.
  • 04
    Payment processor risk
    Stripe could flag custom watch sales as a high-chargeback category (it happens). Mitigation: clear product images, transparent shipping policy, fast refund process, and Affirm/Klarna as backup BNPL options. Worst case, fall back to Shopify Payments which is more lenient on this category.
  • 05
    GDrive photo dependency (proposal phase only)
    The current 4 brand mocks hot-link photos from your Drive. If you change folder sharing, the mocks break. Mitigation: before sending the proposal to anyone external, we download + commit photos to the repo. Already on the punch list.
